Java深拷贝和浅拷贝

发表于 2020-07-09 |
字数统计: 120 | 阅读时长 ≈ 1

复制变量

复制对象

它们指向同一个对象

概念

深拷贝

拷贝出一个全新的对象，与之前无关

浅拷贝

复制了之前对象的引用，并没有开辟新的空间

使用clone方法和直接赋值 都是浅拷贝
Object中本地clone()方法，默认是浅拷贝
如果要深拷贝，要重写clone方法
